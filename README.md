# Feral Forge
A Beast Crushing Game for [Dojo Game Jam VII](https://github.com/dojoengine/game-jams)

## Resources

* [Dojo Starter](https://github.com/dojoengine/dojo-starter) : Initial template with `sozo init`
* [Loot Survivor](https://lootsurvivor.io) | [Savage Summit](https://github.com/Provable-Games/summit) : Beasts, lore, inspiration
* [Pistols at Dawn](https://github.com/underware-gg/pistols) : Project template, rng, hash, leaderboard traits
* [GPT](https://chatgpt.com/) : Background art, ideation
