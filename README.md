# Feral Forge
A Beast Crushing Game for [Dojo Game Jam VII](https://github.com/dojoengine/game-jams)

> In the depths of Death Mountain lies the Feral Forge, a vicious playground where newborn beasts get prepared to rise for battle!

## Resources

* [Dojo Starter](https://github.com/dojoengine/dojo-starter) : Initial template with `sozo init`
* [Loot Survivor](https://lootsurvivor.io) | [Savage Summit](https://github.com/Provable-Games/summit) : Beasts, lore, inspiration
* [1337skulls](https://1337skulls.xyz/) : Original Beasts art
* [Pistols at Dawn](https://github.com/rsodre/feral-forge) : Project template, rng, hash traits, migration scripts
* [Astraea](https://github.com/rsodre/ASTRAEA) : Client template
* [GPT](https://chatgpt.com/) : Background art, ideation
